# Quick Notes

Capture raw, unprocessed information here before triage.

## Input Format
- `YYYY-MM-DD HH:MM - short note`
- `YYYY-MM-DD HH:MM - [tag] short note`

## Triage Rule
After processing an entry, either:
- move it into a destination note, or
- mark it with `triaged` inline.

## Entries
- 2026-01-01 09:00 - [idea] Example capture item.
